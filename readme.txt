== Description ==
WP Survey Manager plugin is a simple extension for creating multiple surveys and questions. And website members can be invited for responding to those surveys. It\'s a simple feedback & survey extension.

== Installation ==
1. Upload and unzip wpsurvey.zip file in plugins directory.
2. Click on activate button link.
3. All done

== Frequently Asked Questions ==
1. Does it support multiple surveys creation & assign to website users?
Answer: Yes, It does support for creating multiple surveys, It also has a question bank module there multiple types questions can be created and added in surveys.

2. How many types of questions can be created using this plugin?
Answer: Below are the different types of questions supported in this plugin.
(1) Text (2) Text Area (3) Radio (4) Checkbox (5) Range Slider

3. Does it support expiry of a survey ?
Answer: Yes, you can specify a expiry date in survey for stopping further responses to be captured in a survey.
